## サービス概要
- 休日のお出かけ先に悩む人へ、気軽に“お出かけ先”を提案するサービスです。
- 実際に行った場所は記録でき、スタンプのように蓄積されます。
- 「行くきっかけ」と「振り返り」をセットで提供し、出不精の解消をサポートします。

## このサービスへの思い・作りたい理由
休日になると「どこへ行こう？」と考えているうちに時間が過ぎ、結局どこにも行かずに終わってしまうことがよくあります。
行きたい場所を探すのが面倒で、気づけば毎週同じ過ごし方になってしまう。そんな“出不精あるある”を自分自身も経験してきました。
そこで、「行き先を提案してくれるだけで、外に出るハードルが下がるのでは？」と思いました。
さらに、行った場所をスタンプのように記録できれば、
- 「せっかくならもう1つ増やしたい」
- 「前より行動できてる」
といった小さな達成感が生まれ、継続的に外出するきっかけになると考えました。

## ユーザー層について
- 出不精気味の社会人
  - 行き先を考えるのが面倒
  - 休日がいつも同じパターンになりがち
  - 外出のきっかけが欲しい
- 近場のお出かけを楽しみたい人
  - 散歩・カフェ巡り・小旅行が好き
  - 新しいスポットを開拓したい
- 行動を記録して達成感を得たい人
  - 写真や日記ほど重くない“軽い記録”が欲しい
  - 行った場所を一覧で振り返りたい

## サービスの利用イメージ
1. アプリを開く
2. 「お出かけ先を提案する」ボタンを押す
3. ランダム or 条件指定でスポットが表示される
4. 気に入ったらそのまま外出のきっかけに
5. 行ったら「行った！」ボタンでスタンプ登録
6. スタンプ帳で自分の行動履歴を振り返る

## ユーザーの獲得について
- SNSで「今日の行き先ガチャ」をシェア
- 行った場所のスタンプ帳を共有できる導線
- ブログやXで「行った記録」を投稿しやすいUI
- 出不精改善・休日の過ごし方に悩む層に刺さるキャッチコピー
  - 例：「休日の行き先、もう悩まない」「外出のハードルを1クリックで下げる」など

## サービスの差別化ポイント・推しポイント
1. 出不精の“行動の最初の一歩”をサポート
検索ではなく“提案”なので、考える負担が少ない。
外出のハードルを下げる設計。
2. 記録がスタンプ形式で“軽く続けられる”
写真や文章ほど重くない。
「行った！」の一押しで記録が残るため、継続しやすい。
3. 行動履歴が可視化され、達成感が得られる
スタンプが増えるほど「もっと行きたい」という気持ちが自然に生まれる。

## 機能候補
MVP（最小機能）
- スポット一覧*
- 行き先のランダム提案
- スポット詳細ページ
- 「行った！」ボタンでスタンプ登録
- 行った場所の一覧（スタンプ帳）
- ユーザー登録・ログイン

本リリースで追加したい機能
- スポットのカテゴリ検索（カフェ・公園・観光地など）

- 行った回数のカウント
- 行った日付のカレンダー表示
- スポットの写真アップロード
- おすすめスポットのレコメンド
- スタンプ帳の共有URL
- 行動ログの可視化（地図表示など）]

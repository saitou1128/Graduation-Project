##サービス概要
本サービスは、山手線を舞台にした“すごろく風お出かけゲーム”です。
サイコロを振って次に向かう駅を決め、駅ごとに用意されたミッションをクリアするとスタンプが獲得できます。
ゲーム感覚で外出のきっかけを作り、行動履歴をスタンプ帳として振り返ることができます。

##このサービスへの思い・作りたい理由
私は普段、休日に「どこへ行こう？」と考えているうちに時間が過ぎ、結局どこにも行かずに終わってしまうことがよくあります。
出不精で行動の最初の一歩が重い自分でも、サイコロで行き先が決まれば悩まずに動けるのではと考えました。
山手線という身近な路線を舞台に、外出しなくてもPC上で遊べる“ゆるいすごろく”として成立しつつ、実際に行けばもっと楽しい体験になるように設計しています。

##ユーザー層について
- 出不精気味で外出のきっかけが欲しい人
- 山手線沿線に住む人、東京が好きな人
- 行動を記録して達成感を得たい人

##利用イメージ
ログイン後、ホーム画面でサイコロを振ると次に進む駅が決まります。
駅ごとのミッションが表示され、クリアするとスタンプを獲得。
履歴画面でで訪れた駅を一覧で確認でき、達成状況が可視化されます。
ゲームとして遊びながら、外出のきっかけにもなる体験を提供します。

##機能候補（MVP）
- サイコロ機能
- 駅データ（固定）
- ミッション表示
- スタンプ付与
- スタンプ帳
- ユーザー登録・ログイン

##使用する技術スタック
- フレームワーク：Ruby on Rails
- データベース：PostgreSQL
- デプロイ先：Render
